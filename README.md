## Features

### Questions
- **163 questions** covering the full CCNA 1 v7.0 final exam syllabus
- Questions appear in **original document order** with original question numbers shown (e.g. `Original #45`)
- **40+ exhibit images** embedded directly — network diagrams, CLI output, topology maps, cable photos
- **Multi-answer questions** flagged with a yellow `Choose multiple` badge
- **Case variant questions** — Q55 (LLC sublayer) and Q77 (MAC sublayer) each appear as multiple scenario variants (5 and 4 cases respectively) so you practise all possible option combinations

### Per-Option Explanations
After answering, every single option reveals an inline explanation:
- ✓ **Green** — what this is and exactly why it is correct
- ✗ **Red** — what this is and why it is wrong
- **Grey** — what this option represents (even if you did not select it)

A **📖 Topic Summary** panel also appears below each answered question covering the broader concept.

### Memorise Questions
Some questions are image/matching type with no selectable options — these auto-display the answer and are labelled **📌 Memorise**. They are completely excluded from your score, matching how the real Cisco exam works.

Questions treated as Memorise (not scored):
`Q12, Q14, Q17, Q33, Q58, Q96, Q99, Q101, Q110, Q120, Q126, Q145, Q149, Q152, Q163`

### Scoring
- Calculated as `(correct answers ÷ scored questions) × 100`, rounded to the nearest integer
- Memorise questions are **excluded** from the denominator — exactly like the real CCNA exam
- Passing threshold shown as **≈83%** (the approximate real Cisco CCNA passing score)
- Results broken into four buckets: **Correct / Wrong / Skipped / Memorise**

## Quiz Modes

| Mode | Description |
|---|---|
| **All 163** | Every question in original order — full exam simulation |
| **Random 60** | 60 randomly selected questions — good for a study session |

| **🔁 Practice Wrong** | Only questions you got wrong, skipped, or memorised in your last run — unlocks after completing any quiz |

## Navigation

| Control | Action |
|---|---|
| **Check Answer** | Reveal correct/wrong options and all explanations |
| **Next →** | Advance to the next question |
| **← Back** | Return to the previous question (removes last result) |
| **Skip** | Show the correct answer without scoring |
| **CCNA 1** (top-left badge) | Return to the main menu from anywhere |

---

## Score Screen

After completing a quiz you see:
- A percentage ring coloured green (≥83%), yellow (≥65%), or red (below 65%)
- Breakdown of Correct / Wrong / Skipped / Memorise counts
- A note showing exactly how many questions were scored
- **Review Wrong** — click any item to expand the correct answer and topic summary
- **🔁 Practice Wrong** — jump straight into a practice session of only your missed questions

## Content Coverage

Topics drawn from the CCNA 1 Introduction to Networks (ITN) v7.0 curriculum:

- Network models (OSI, TCP/IP) and layer functions
- IPv4 and IPv6 addressing, subnetting, VLSM
- Network protocols — HTTP/S, DNS, DHCP, FTP, SMTP, POP3, IMAP, SSH, Telnet, TFTP, SNMP, ARP, ICMP, NDP
- Transport layer — TCP vs UDP, ports, windowing, handshake, multiplexing
- Data Link layer — Ethernet, MAC/LLC sublayers, CSMA/CD, CSMA/CA, frame structure, FCS
- Physical layer — UTP, STP, fibre optic, connectors, attenuation, EMI/RFI
- Network security — viruses, worms, Trojans, DoS, phishing, spyware, AAA framework
- Switching — store-and-forward, cut-through, VLANs, SVI
- Routing fundamentals — default gateway, routing table, TTL, NAT
- IPv6 — global unicast, link-local, multicast, SLAAC, EUI-64, NDP
- CLI — Cisco IOS privilege modes, SSH configuration, banner, startup-config vs running-config
